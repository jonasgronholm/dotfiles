# SDDM Login Manager

Using simple_sddm_2 theme.

## Restore

1. Install SDDM: `sudo pacman -S sddm simple-sddm-2`
2. Enable SDDM: `sudo systemctl enable sddm`
3. Copy theme.conf to `/usr/share/sddm/themes/simple_sddm_2/theme.conf`
4. Copy your wallpaper to `/usr/share/sddm/themes/simple_sddm_2/Backgrounds/wallpaper.png`
